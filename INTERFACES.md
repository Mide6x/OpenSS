# OpenSS Interfaces

Available entry points:

- `openssmide`  
  Shows the welcome screen and available commands.

- `openssmide capture`  
  Capture the active Chrome window and analyze it.

- `openssmide voice`  
  Record a short voice question and get an AI response.

- `openssmide chat`  
  Resume a previous session and continue the conversation.

- `openssmide history`  
  List recent sessions.

- `openssmide update`  
  Pull latest code and update dependencies.
